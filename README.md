## Main Purpose

#### FasalBridge allows users to:
- Post what the are growing or selling.
- Browse other user's crop posts.
- Show interest to connect and collaborate.
#### The platform focuses on community interaction in the agricultural sector than traditional buyer-seller transactions.

## Features

### Navbar

#### The application includes a responsive navigate bar with:
- Home
- All Crops
- My Posts
- Profile
- My Interests

### Home Page

- Hero Slider: Showcasing featured crops.
- Latest Crops Section: Display recent crop posts.
- How It Works Section: Users can post crops, browse other's posts, send interest requests and manage interactions between buyers and crop owners.
- Agro Blogs: Educational content on agriculture.
- Happy Farmers Section: Success stories of farmers.
- Agriculture Experts Section: Guidance from agricultural experts.

### Crop Management

- Add, edit and delete crops.
- View crop details with full information.
- Track crop quantity automatically based on accepted interests.

### Interactions:

- Owners can accept or reject crop interests on the crop details page.
- Quantity updates automatically when an interest is accepted.
- Users can track their sent interests on the My Interests page.
- Interests display crop name, owner, requested quantity, message and status.

### UI/UX Enhancements
- React icons for intuitive visuals.
- Framer Motion for smooth animations.
- React fast Marquee for scrolling highlights.
- Tailwind CSS & DaisyUI for responsive, modern UI.
- React Hot Toast & SweetAlert for notifications and alerts.

### Authentication
- Firebase Authentication secures user login and signUp.
### Backend 
- Built with Node.js, Express.js and MongoDB.
- Provides full CRUD operations for crops and user data.
- Routes are protected with Firebase SDK.

### Technologies Used
- Frontend: React.js, Tailwind CSS, Framer Motion, React Fast Marquee, React Router, React Icons
- Backend: Node.js, Express.js, MongoDB
- Authentication: Firebase
- Notifications: React Hot Toast, SweetAlert
